  5. A data type is different than its value. Some programming languages are strongly typed, requiring programmers to predefine what kind of data they are using in variables, parameters, functions, etc. before they actually use them. Other loosely typed languages like Javascript, Perl, and PHP, do not require programmers to define their type ahead of time. Booleans, integers, floats and strings are common primitive data types that can be found across many programming languages. Null and undefined are types that represent no data. Most programming languages allow you to convert data types, this conversion can also happen automatically which is known as “implicit conversion”. To add on, there are also other types known as arrays, objects, lists and etc.
